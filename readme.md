# Tic Tac Toe Game

A modern, interactive Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

✨ **Modern UI Design**
- Sleek dark theme with blue and orange colors
- Smooth animations and transitions
- Responsive design for mobile and desktop

🎮 **Gameplay**
- Two-player local game (X vs O)
- Real-time turn indicator
- Win detection with visual feedback
- Draw detection
- Score tracking across multiple games
- Reset button to start new game

🎨 **Design Elements**
- Animated game board with hover effects
- Glowing symbols (X and O)
- Dynamic status indicator
- Player info cards with active state
